HoverMenu

Uses JavaScript (a jQuery plugin) and CSS to implement dropdown menus.

Why JavaScript? To allow for a slight pause before the dropdown closes. Also, to prevent more than one dropdown being open at once. It’s more about the hovering behavior than the dropdown behavior. Thus “HoverMenu.”

To use: Simple. Include the JavaScript and CSS files (see below).
The HTML should look something like: 

<nav id="main">
  <ul>
    <li>Item 1</li>
    <li>
      Item 2 (with submenu)
      <ul>
        <li>Item 2a</li>
        <li>Item 2b</li>
      </ul>
    </li>
  </ul>
</nav>

Note the nested <ul>s.

And then: 

  $(function() { // Document ready
    $('nav#main').hoverMenu();
  });

$.fn.hoverMenu accepts an optional options hash. Available options are:

  • topLinkSelector defaults to "a" — The selector for the parent "link" that hides the nested menu. Assumed to be an "a", but, in case it isn’t a link, may be "span.nolink", etc.

Example (try hovering):

  • Link 2 (with submenu)

    • Link 2b
  • Link 3 (more submenus)
